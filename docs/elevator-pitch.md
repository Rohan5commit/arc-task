# ArcTask elevator pitch

ArcTask turns a single goal into a micro-economy.

A coordinator agent decomposes the goal into atomic tasks, assigns each task a nano-payment in USDC terms, routes the work across agents or humans, verifies outputs, and writes payout receipts into an Arc-ready ledger.

The demo is intentionally simple: no auth, no protocol sprawl, just one screen that makes the economics of agentic work visible. That is why Arc matters. Stablecoin-denominated fees and fast deterministic settlement make tiny programmable payouts practical instead of theoretical.
